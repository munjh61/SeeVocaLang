# 씨보랭 (See Voca Language)


**사진 인식 기반 영어 단어 학습 웹앱**

AI 객체 탐지 기술을 기반으로 사용자가 직접 촬영하거나 업로드한 사진 속 물체를 인식하여

**영어 단어를 알려주고 학습할 수 있는 서비스**입니다.

영어를 잘 모르는 사용자를 위해 직관적인 UI, 반복 학습 게임, 친구 단어장 보기 기능을 통해

**일상 속에서 친숙하고 재미있게 영어 단어를 익힐 수 있도록 설계**되었습니다.

## ⚙️ 기능

### 메인 기능

- **AI 사진 객체 인식 기반 단어 학습**
    - 사용자가 촬영·업로드한 사진을 **YOLOv8 모델**로 객체 인식
    - 인식된 객체를 **영어 단어와 매핑**하여 제공
    - 기초 영어 학습자가 **일상 속 사물**로 자연스럽게 단어 학습
- **나만의 단어장**
    - 인식된 단어를 **개인 단어장에 저장**
    - TTS(Text-to-Speech)로 단어 발음 지원
- **게임·퀴즈 복습 기능**
    - 개인 단어장을 기반으로 다양한 **게임·퀴즈** 제공
    - 학습자의 **재미와 학습 효율**을 동시에 강화
- **오늘의 학습 & 학습 통계**
    - 매일 새로운 단어 추천
    - 연속 학습 일수와 학습 완료일 캘린더 등 **통계 시각화**

### 부가 기능
- **소셜 로그인**
    - **Naver / Kakao** 계정 로그인 지원
    - 최초 로그인 시 소셜 닉네임 자동 반영
- **이메일 인증 & 임시 비밀번호 발급**
    - 이메일 인증 코드 발송 및 검증
    - 임시 비밀번호 발급 후 재설정 가능
- **프로필 관리**
    - 닉네임, 프로필 이미지 수정
    - 계정 탈퇴 (Soft Delete 처리)
- **단어장 즐겨찾기 & 폴더 관리**
    - 원하는 단어장을 **즐겨찾기** 등록
    - 단어장 폴더 생성 및 관리
- **단어장 공유 기능**
    - 단어장을 친구와 공유
    - 협력 학습 환경 제공

## 🧑‍🎓 페르소나

### (1) 저소득층 초등학교 1학년

- 선행학습 기회가 없어 영어에 대한 기초 지식이 없음.
- 친구들이 하는 영어 단어 빙고 게임에 흥미와 소외감을 느끼며 참여하고 싶어함.
- 주변 물건 사진을 찍어 단어를 익히고 친구들과 함께 게임을 즐기고자 함.

### (2) 70대 초반 평생교육원 수강생

- 늦은 나이지만 영어를 배우고자 하는 열정이 있음.
- 복잡한 교재보다 실생활에서 익숙한 물건을 통해 배우는 방식에 적응이 빠름.
- 쉽고 명확한 UI, 반복 가능한 게임 중심 학습을 선호함.

### (3) 중졸 학력의 60대 여성

- 일상 속 영어 간판이나 메뉴판 해석에 어려움을 느끼며 불편함을 겪고 있음.
- 디지털 환경에 익숙하진 않지만, 자녀가 알려준 사이트를 통해 편하게 영어를 접하고자 함.
- 텍스트 중심보단 이미지·음성 중심의 학습 형태를 더 선호함.

## 문준호
### 단어장 페이지
![단어장](/asset/3.%20단어장%20페이지/word%20(1).png)
1. 단어장 CRUD
    ```
    - 단어장을 불러올 수 있다.
    - 단어장을 생성할 수 있다.
    - 단어장을 수정할 수 있다.
    - 단어장을 삭제할 수 있다.
    ```
2. 즐겨찾기
    ```
    - 단어장 즐겨찾기를 할 수 있다.
    - 단어장 즐겨찾기를 취소할 수 있다.
    - 단어장 즐겨찾기된 기준으로 필터링할 수 있다.
    ```
3. 전체 단어
    ```
    - 모든 단어장의 단어를 볼 수 있다.
    - 해당 단어가 어디에 저장되어있는 지 라벨링한다.
    - 라벨을 클릭하면 해당 단어장 상세 페이지로 이동한다.
    - 단어장 삭제시 단어도 라벨링 처리 또는 라벨링이 0개가 되는 경우 삭제처리.
    - 퀴즈를 풀 단어장을 선택할 수 있다.
    ```
4. 검색창
    ```
    - 단어장 이름 또는 단어를 기준으로 검색할 수 있다.
    - 검색마다 백엔드에 연결할 필요 없이, 페이지 접속 시 데이터를 가져오고 필터링한다.
    - hangul-js를 사용하여 한글도 검색이 유용하도록 한다.
    ```

## 단어장 상세페이지
![단어장상세](/asset//3.%20단어장%20페이지/word%20(2).png)
1. 단어장 내 단어 표시
    ```
    - 특정 단어장에 저장된 단어 정보를 불러올 수 있다.

    - 버튼을 눌러 발음을 들을 수 있다.

    - 영문, 한글 기준으로 단어 검색 할 수 있다.
    - 버튼을 누르면 영문, 한글 단어을 일시적으로 가릴 수 있다.
    ```

### 퀴즈 페이지
![퀴즈](/asset/4.%20퀴즈%20페이지/quiz%20(1).png)
1. 디자인
    ```
    - 상단: 현재 단어장 이름, 단어장 부제목, 현재 문항/총 문제수를 표시한다.
    - 중단: 현재 문항의 이미지를 표시한다.
    - 하단: 버튼 8개를 표시한다. 버튼은 현재 문항의 영문이 적힌 버튼 1개, 그리고 다른 문항의 영문 버튼 7개로 이루어져있다.
    ```
2. 퀴즈 생성
    ```
    - 단어장에 저장된 단어 또는 오늘의 학습 단어 목록을 불러와 퀴즈를 생성한다.
    - 오늘의 학습인 경우 매번 문제를 풀 때마다 현재 진행도를 저장한다.
    ```
3. 퀴즈 버튼
    ```
    - 현재 이미지에 맞는 단어를 클릭한 경우 다음 동그라미 표시를 하고 다음 문제로 넘어간다.
    - 현재 이미지와 틀린 단어를 클릭한 경우 가위 표시를 하고 다음 문제로 넘어가지 않는다.
    ```
4. 모든 문제를 풀면 퀴즈 완료 페이지로 이동한다.

### 게임 페이지 3 - 폭탄 피하기
1. 오늘의 단어에 수록된 단어들이 폭탄으로 구성됨.
2. 게임 방법
    ```
    1. 이미지가 부착된 폭탄이 우측으로 도달하면 폭탄 소리와 이미지가 일시적으로 출력
    2. 문제를 맞출 수록 폭탄의 속도가 10% 빨라짐.
    3. 8문제를 맞출 때마다 round가 올라가며 round 마다 좌측의 적 해적의 이미지가 달라짐
    4. 5번 폭탄에 맞을 경우 게임 종료
    5. round 3 이상일 경우 승리 모달 표시, 미만은 패배 모달 표시
    ```
